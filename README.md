# emotion diary
이 프로젝트는 노마드 코더의 Netflix clone 강의를 보며 공부한 내용과 간단한 추가적인 기능을 넣었습니다. <br />
추가 기능으로 React-hook-form을 연습하고자 회원가입과 로그인를 추가했습니다. <br />

재사용에 대해 고민하며 효율적으로 컴포넌트 분리를 할 예정입니다. <br />
